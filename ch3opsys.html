Operating Systems Terminology
Term Definition
operating system Software that controls the overall operation of a computer.
job queue storage organization in which objects are ordered using first-in, first-out.
FIFO First in, first out. Objects are removed in the order they arrived.
interactive processing program can carry on a dialogue with the user through remote terminals.
time-sharing providing service to multiple users simultaneously.
multiprogramming dividing time into intervals and assigning intervals to jobs, creating the illusion of multiple programs running simultaneously.
application software programs for performing tasks particular to the machine's utilization.
system software programs performing tasks common to computer systems in general.
utility software software fundamental to computer installations, but not included in the operating system.
graphical user interface (GUI) interface using pictoral representations of objects, generally interacted via mouse, stylus, or touchscreen.
shell text-based user interface utilizing the keyboard and monitor.
kernel internal part of an operating system, containing the most basic components, such as a file manager.
device driver definition here.
file manager coordinates the machine's mass storage via records of storage locations and access permissions of files.
memory manager coordinates usage of memory so tasks do not encroach on each others' memory allocation.
virtual memory fictional memory exceeding the physical memory, achieved by swapping pages of memory between mass storage and the main memory.
boot strapping procedure transferring the operating system from mass storage to main memory by using ROM contents to refresh the main memory.
ROM Read-Only Memory. Cannot be altered, only read.
deadlock condition in which two or more processes cannot complete as they are waiting for resources allocated to each other.
spooling technique to avoid deadlocks by giving the impression a resource is available, but holding the data in mass storage to be processed when the resource is available.
super user (or administrator) user with highly privileged access; able to modify operating system settings.
multi-factor authentication multiple methods of authentication are required to gain access to a system protected by multi-factor authentication.
