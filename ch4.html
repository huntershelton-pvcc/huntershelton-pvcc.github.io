Chapter 4: Networking

Hunter Shelton

Item Definition
Network Classifications
  1. LAN
  2. MAN
  3. WAN
  1. Local Area Network (single building)
  2. Metropolitan Area Network (city)
  3. Wide Area Network (wider area/multiple cities)
Network Models
  1. bus network
  2. star network
  1. a network in which all the machines are connected to a common communication line called a bus
  2. a network in which one machine serves as the central focal point to which all the others are connected
Protocols
  1. CSMA/CD Ethernet
  2. CSMA/CA Wifi
  1. Multiple Access with Collision Detection
  2. Multiple Access with Collision Avoidance
Combining networks:
  1. repeater
  2. bridge
  3. switch
  4. router
  5. gateway
  1. repeats signals between buses
  2. connects buses and only passes signals destined for the other side
  3. multi-connection bridge
  4. special-purpose computers forwarding messages between networks
  5. router communicating with the internet
Client/server model
  1. P2P: peer-to-peer
  2. distributed system
  3. cloud computing
  1. peers communicate as equals on a one-to-one basis
  2. software executing as processed on different computers
  3. huge pools of shared computers on the network allocated for use as needed
Internet Addressing
  1. Tier 1
  2. Tier 2
  3. Tier 3
  1. Internet Service Provider for high-speed, high-capacity, international WAN
  2. Regional network providing redundancy
  3. Intranet supplying internet access to individual homes/businesses
IP Address
  1. ICANN
  2. DNS
  3. TLD: Top-level domain
  4. Country-code TLDs
  1. Internet Corporation for Assigned Names and Numbers. Coordinates internet operation and assigns IP address blocks to ISPs.
  2. Domain Name System. Translates domain names into IP addresses via directory lookup.
  3. Reflects the domain classification
  4. Two-letter TLDs for specific countries
Internet protocols
  1. FTP
  2. HTTP
  3. SMTP
  1. File Transfer Protocol
  2. Hyper Text Transfer Protocol
  3. Simple Mail Transfer Protocol
World Wide Web
  1. URL
  2. client-side activities
  3. server-side activities
  1. Uniform Resource Locator
  2. performed on a client (browser)
  3. performed by a server, such as a webserver
Internet four-layer software
  1. application layer
  2. transport layer
  3. network layer
  4. link layer
  1. software using internet communication to carry out tasks
  2. divides messages into distinct packets, reassembled at the destination
  3. decides the route to forward packets through the internet
  4. responsible for transferring and receiving the packet at the device level
Internet is:
  1. packet-switching network
  2. fault tolerant
  3. redundant
  1. packets are sent through switches in a network. packets contain data transmitted and control info for routing the data
  2. can deal with mistakes and errors
  3. has fallback pathways for delivery in the case of congestion or failure
Cybersecurity
  1. malware
  2. virus
  3. worm
  4. Trojan horse
  5. spyware
  6. phishing
  7. DDoS attack
  1. malicious software
  2. program that replicates itself by inserting itself into another program
  3. autonomous program transferring itself through a network, forwarding copies of itself to other computers
  4. malware disguised as a desirable program. often sets a delayed payload
  5. software collecting information about the computer it resides upon
  6. technique of obtaining information simply by asking for it
  7. distributed denial-of-service attack
Protection
  1. firewall
  2. spam filter
  3. proxy server
  4. antivirus software
  5. cryptography
  6. public key encryption
  1. filters traffic through a point in a network
  2. firewall blocking unwanted email
  3. software unit acting as an intermediary between client and server
  4. detects and removes known viruses and other infections
  5. study of sending messages securely in the presence of adversaries
  6. a known public key encrypts messages, and a secret private key is used to decrypt messages
