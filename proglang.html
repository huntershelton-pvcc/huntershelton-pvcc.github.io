Chapter 6: Programming Languages

Hunter Shelton

Item Definition
machine language sequences of instructions encoded as numeric digits
debugging the tedious task of locating and correcting errors
assembler programs which convert mnemonic expressions into machine language instructions
assembly language mnemonic expression system for representing programs used by assemblers
translator translates programs expressed in high-level primitives into machine-language programs
compiler also known as translation programs. another word for translator, as it compiles several machine instructions to simulate high-level primitives
interpreter alternative to translator that executes a program from its high-level form rather than translating to machine language
cross-platform software software independent of an operating system and machine design, therefore executable throughout a network
natural language language evolved over time without formal grammatical analysis
formal language language that is precisely defined by grammars
machine independence language that does not rely on the characteristics of a particular machine
high-level programming language provide more abstractions for the programmer and make it easier for people to read and write programs
object-oriented programming (OOP) programming process where a software system is viewed as a collection of units, called objects, each of which is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects
scripting language subset of the imperative programming languages
data types:
  1. integer
  2. float
  3. string
  1. numeric value consisting of whole numbers
  2. numeric data containing values other than whole numbers, stored in floating-point notation
  3. multiple characters in sequence, generally terminated with a null byte
primitive data type data types included as primitives in a programming language, such as char, int, and bool
data structure conceptual shape or arrangement of data
Python collection data structures
  1. list
  2. tuple
  3. set
  4. dictionary
  1. A collection which is ordered, changeable, and allows duplicates. Created using square brackets.
  2. A collection which is ordered and unchangeable. Written with parentheses.
  3. A collection which is unordered, unchangeable, and unindexed. Delineated with curly braces.
  4. A collection which is ordered, changeable, and does not allow duplicates. Delineated with curly braces using key: value format.
function A set of instructions for performing a task that can be used as abstract tool by other program units.
local variable A variable only able to be references within a function.
global variable Varables with unrestricted scope. They can be accessed globally.
source program original form of a program
object program translated form of a program
