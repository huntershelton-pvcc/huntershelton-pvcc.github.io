All About Github

What is a version control system?

  • A version control system tracks the history of changes made to a project by collaborators.
  • Users can view project history to see changes to files over time.
  • Previous revisions of a project can be recovered from past commits.

What is Git?

  • A free, open-source version control system developed by Linus Torvalds.
  • It is designed to handle both small and large projects quickly and efficiently.

What is Github?

  • Cloud-based git hosting platform.
  • Developers can collaborate with others and commit to existing projects.
  • Users can review changes and commit to projects.

What is the difference between Git and Github?

  • Git is the version control system, while Github is an platform for hosting git projects.

Who started Github and how was it started?

  • Developement started October 19, 2007 and it launched April 2008.
  • It was originally founded as Logical Awesome.
  • Developed by: Tom-Preston Werner, Chris Wanstrath,
    P.J. Hyett and Scott Chacon.
  • It was developed using Ruby on Rails.

What company owns it now?

  • Microsoft.

How much does a Github account cost?

  • A basic GitHub account is free.
  • A team account is $4/month.
  • Enterprise is $21/month.

What is the Octocat?

  • Github's mascot, Octocat, is a fusion between a cat and an octopus.
  • It has five arms.
  • Its name is Monalisa, inspired by a story a child wrote about Octocat.
Word Definition
repository Virtual storage space for files and code.
Commonly abbreviated to "repo".
commit Record of changes to a set of files.
fork Creates a copy of the repository tied to your account containing the repository's current commit history.
push Uploading from a local repository to a remote repository.
pull request Request to merge a set of changes from one branch to another.
Collaborators can review proposed changes for the target code branch.
workflows Automated process running one of more jobs in a repository. Configurable by YAML.
issues Tickets created in a repository to plan, discuss, and track work.
raw button Allows users to view the raw content of a file without any styling.
blame button Shows the commit last modifying specific lines.
